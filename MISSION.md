# Mission: Agentic Coding and React Development

## Why
Prepare for and successfully clear an upcoming technical interview targeting Agentic AI workflows and React development by learning React concepts fresh from the ground up, complemented by building a hands-on project step-by-step.

## Success looks like
- Confidently explain Agentic Coding concepts (LLMs, Skills, Rules, MCP servers, and coding harness leverage).
- Write clean React components from scratch using JSX, handling state and props correctly.
- Implement React Hooks (including custom hooks, memoization, forwardRef).
- Describe and implement different state management strategies (useReducer, Redux Toolkit, Jotai) within a practical project.
- Apply styling in React (CSS Modules, SCSS, Styled Components) and configure Storybook stories.
- Identify and fix React performance bottlenecks (unnecessary re-renders, code splitting, lazy loading).
- Write React unit tests and describe high-level E2E testing strategy.

## Constraints
- Focused, interview-oriented learning.
- Teach React concepts fresh (from the start) without relying heavily on Angular or Vue comparisons.
- **Hands-on practice**: Build a small project step-by-step. The agent provides instructions, code structure, and design hints, and the user writes the code.
- Time-constrained preparation; priority on high-yield, interview-frequent concepts.

## Out of scope
- Full-stack database/backend development (except MCP server integration details).
- Extensive comparative deep-dives into Vue/Angular.
