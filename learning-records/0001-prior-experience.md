# Prior Experience and Learning Profile

The learner has established their baseline technical background and core preparation goals:
- **Prior Framework Experience**: Strong working knowledge of Angular, and basic familiarity with VueJS. This indicates they understand components, single-page application (SPA) routing, lifecycle hooks, state management patterns (like NgRx or Vuex), and declarative rendering.
- **React Background**: Zero hands-on React experience. They have never used it, meaning we need to build React from the ground up, specifically focusing on how React's model differs from Angular's change detection and Vue's template reactivity.
- **Agentic Coding**: The learner wants to understand how to build and leverage agentic workflows, custom skills (`Skills.md`), workspace rules (`Agents.md`), and Model Context Protocol (MCP) servers.
