Agentic Coding Reference

Quick reference guide for customizing AI coding agents, managing project rules, defining custom skills, and using MCP servers.

1. Rules (AGENTS.md)

Purpose: Appends persistent, project-wide or global behavioral constraints directly to the agent's base instructions.

Locations:

  • Global: C:\Users\<username>\.gemini\config\AGENTS.md
  • Workspace: <project_root>\.agents\AGENTS.md

Best For: Code styling, architecture standards, testing levels, security policies, and tech-stack choices.

2. Skills (SKILL.md)

Purpose: Procedural, on-demand modules containing checklists, instructions, and scripts for specific tasks.

Structure:

skills/<skill_name>/
├── SKILL.md (Frontmatter + Guide)
├── scripts/ (Helper scripts)
├── examples/ (Code snippets)
└── resources/ (Assets or docs)

Matching: Triggered via description/name match. Exposes files/tools to the agent.

3. SKILL.md Frontmatter Fields

  • name (String): Unique identifier. Matches commands and triggers.
  • description (String): Short summary for matching search intent.
  • argument-hint (String): Placeholder for argument hint in CLI / UI.
  • disable-model-invocation (Boolean): Disables direct AI call, routing through custom logic instead.

4. Model Context Protocol (MCP)

Purpose: Open standard that enables AI models to connect securely to local/remote tools, data sources, and services.

Core Capabilities:

  • Tools: Executable functions (e.g., compile code, search DB, query APIs).
  • Resources: Semi-static data sources (e.g., database schemas, logs, API docs).
  • Prompts: Pre-built templates/workflows for the agent to reuse.

Best Practices for Writing System-Level Instructions

  • Clear Precedence: When conflicts occur between global rules and workspace rules, workspace rules override.
  • Be Actionable: Write rules as specific constraints ("Do not use X") or step-by-step procedures rather than general observations.
  • Write for the Parser: Maintain well-structured YAML frontmatter in SKILL.md. A missing line or incorrect key will prevent the agent from discovering it.
