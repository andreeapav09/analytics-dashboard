Reconciliation & Fiber Engine

React uses an internal reconciliation engine called React Fiber. Fiber allows React to split rendering work into incremental chunks, pause work, and resume it later. This keeps the browser main thread responsive during heavy updates.

Rules of JSX

  1. Return a single root element: Wrap content in a parent element or a Fragment (<>...</>).
  2. Close all tags: Every tag must be explicitly closed (e.g., <br />, <input />).
  3. camelCase everything: Use camelCase for attributes (e.g., className instead of class, onClick instead of onclick).
