# Agentic Coding and React Development Resources

## Knowledge

### Agentic Coding & AI Harnesses
- [Documentation: Model Context Protocol (MCP) Official Site](https://modelcontextprotocol.io)
  Official specification for MCP servers and client integrations. Use for: understanding MCP server design, transport protocols, and tools/resources/prompts definition.
- [Skill Reference: Antigravity Guide (Built-in)](file:///C:/Users/z0043xbv/.gemini/antigravity/builtin/skills/antigravity_guide/SKILL.md)
  The built-in documentation for Google Antigravity features, CLI commands (`agy`), configuration rules (`Agents.md`), and custom skills (`Skills.md`).
- [Guide: Agentic Design Patterns - Andrew Ng (DeepLearning.AI)](https://www.deeplearning.ai/the-batch/how-ai-agents-can-improve-productivity-and-problem-solving/)
  Conceptual overview of Agentic design patterns (Reflection, Tool Use, Planning, Multi-agent collaboration).

### React Fundamentals & Hooks
- [Official Site: React Documentation (react.dev)](https://react.dev)
  Canonical reference for all hooks (`useState`, `useEffect`, `useMemo`, `useCallback`, `useRef`, `useReducer`, `forwardRef`).
- [Article: React 19 Upgrade Guide](https://react.dev/blog/2024/12/05/react-19)
  Detailed overview of new React 19 features (Actions, `use` hook, server components, form helpers).
- [Article: Virtual DOM and React Fiber Architecture](https://github.com/acdlite/react-fiber-architecture)
  A description of React's reconciliation algorithm and the Fiber engine.

### State Management
- [Official Site: Redux Toolkit Documentation](https://redux-toolkit.js.org)
  Guide to modern Redux, Slices, store setup, and RTK Query.
- [Official Site: Jotai Documentation](https://jotai.org)
  Atomic state management concepts, Atoms, and comparison with Redux/Context.

### Storybook & Testing
- [Official Site: Storybook Documentation](https://storybook.js.org/docs)
  Setup, writing CSF (Component Story Format) stories, args, and addons.
- [Official Site: React Testing Library (RTL)](https://testing-library.com/docs/react-testing-library/intro/)
  Unit testing components from a user-centric perspective.
